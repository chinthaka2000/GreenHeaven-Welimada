# greenheaven Welimada - Agro-Tourism Web Application

**Live Demo**: [https://greenheaven-welimada.onrender.com](https://greenheaven-welimada.onrender.com)

greenheaven Welimada is a premium, responsive Single Page Application (SPA) designed for an eco-tourism sanctuary in the lush hills of Welimada, Sri Lanka. The platform provides users with a seamless, immersive experience to explore farm retreats, accommodations, and local experiences while maintaining high performance and modern aesthetics.

## Key Features
- **Modern UI/UX**: Designed to convey a calm, natural, and premium "earthy" vibe, featuring smooth animations and a refined typography system.
- **Client-Side Routing**: Implemented using React Router to ensure fast, seamless navigation across the entire application without full page reloads.
- **Backend Integration**: Utilizes Supabase (Backend-as-a-Service) to manage secure newsletter subscriptions and handle administrative authentication workflows.
- **Fully Responsive**: Optimized for all devices, ensuring an elegant layout on both mobile and desktop screens.
- **Edge Deployment Optimized**: Configured with custom routing rules (via `vercel.json`) to flawlessly serve the SPA architecture across edge networks without routing conflicts.

## Tech Stack
- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Tailwind CSS, Framer Motion (for animations), Lucide React (for icons)
- **Backend/Database**: Supabase
- **Deployment**: Vercel / Render

## Getting Started

To run this project locally:

1. Clone the repository
2. Run `npm install` to install dependencies
3. Set up your `.env.local` file with the required Supabase credentials
4. Run `npm run dev` to start the Vite development server
